### Setup

##### 1.

```
pip install langchain
pip install openai
```
(You need python installed)


##### 2.

```
export OPENAI_API_KEY="..."
```

(Get your API key from: https://platform.openai.com/account/api-keys)

### Run 

```
python3 dostuff.py
```
